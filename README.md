# A/B Testing Analysis Project

## Objective
To determine whether a new website design improves user conversion rates.

## Dataset
Synthetic dataset of 10,000 users divided into:
- Control group (old design)
- Treatment group (new design)

## Methodology
- Data cleaning and exploration
- Conversion rate analysis
- Two-proportion Z-test

## Results
- Control Conversion Rate: ~10%
- Treatment Conversion Rate: ~12%
- P-value: 0.000114

## Conclusion
The treatment group shows a statistically significant improvement in conversion rate.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
